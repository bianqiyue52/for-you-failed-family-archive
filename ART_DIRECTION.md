# Art Direction

## Direction

The artwork should feel like a failed family archive interface / classification machine.

It begins with order: documents, grids, forms, records, labels, categories, and administrative confidence. Interaction should not turn the work into a dirty ruin. It should reveal order breaking down through paperwork logic that cannot contain the family event.

## Visual Keywords

- household register page
- school district zoning map
- floor plan
- evidence table
- spreadsheet grid
- administrative form
- redaction
- crossed-out fields
- misfiled records
- water-damaged paper
- cut waveform
- unresolved category

## Avoid

- haunted house
- dirty ruin
- horror atmosphere
- generic childhood trauma aesthetic
- decay as the main visual language
- nostalgic yellow old-house filter

## Interaction Feel

The interface should look over-organized at first. With repeated interaction, categories should fail, labels should drift, fields should contradict each other, and traces should feel like archival damage: redaction, correction, misfiling, and failed evidence handling.
